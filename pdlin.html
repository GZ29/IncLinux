Notamment grâce à son modèle open source (code source réutilisable), Linux a permis de donner naissance à des centaines
d'OS ou distributions Linux, c'est-à-dire de systèmes d'exploitation complets basés. Ces distributions peuvent être conçues
pour jouer le rôle d'OS sur un PC, un objet connecté, un serveur...
______
Parmi les distributions Linux les plus populaires figurent, notamment, Linux Mint, Linux Ubuntu, Red Hat, Debian, Suse Linux,
ou encore Kali Linux et Arch Linux. Chacune a ses propres spécificités (voir le tableau ci-dessous).
______
Les distributions Linux les plus populaires
Nom Site Web Editeurs Commentaire
Linux Debian Debian.org Projet Open Source Distribution Linux pour serveur et ordinateur. Cette distribution est notamment connue pour avoir été retenue par la ville de Munich pour équiper 14 000 ordinateurs.
Linux Ubuntu Ubuntu.com Canonical Distribution Linux pour ordinateur et serveur reposant sur la distribution Debian. Elle a aussi donné lieu à des déclinaisons pour TV (Ubuntu TV), smartphone (Ubuntu Touch), etr IoT (Snappy Ubuntu Core)
Linux Mint LinuxMint.com Projet open source Distribution Linux pour ordinateur, basée sur Ubuntu. Linux Mint se différencie d'Ubuntu via un gestionnaire de bureau qui lui est propre (Cinnamon).
Red Hat / Fedora Red Hat Enterprise/GetFedora.org Red Hat Distribution Linux pour serveur. Son éditeur Red Hat est le principal pure player des systèmes serveur open source. Il enregistre un chiffre d'affaires de 2 milliards de dollars.
Suse Linux Suse.com Micro Focus Distribution Linux qui se décline à la fois pour PC et pour serveur. Il s'agit de la plus ancienne distribution Linux commerciale encore existante. Après avoir été la propriété de Novell puis d'Attachmate, elle appartient à Micro Focus depuis 2014.
Kali Linux Kali.org Projet open source Distribution Linux conçue pour l'audit et le test d'intrusion. Elle repose sur Debian.
Arch Linux Archlinux.org Projet open source Distribution Linux dessinée pour les utilisateurs Linux avancés, les puristes. Se voulant très légère, elle est livrée sans outil graphique et implique donc des compétences techniques. Elle peut s'installer aussi bien sur un PC qu'un serveur.
